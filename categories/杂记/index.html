Hello Hexo

May 10 2014

以前自己记录的知识多在印象笔记、硬盘了等等的零零散散的地方，今天算是把之前的东西准备规整到github page上。

感谢万能的github

这个静态博客是基于Hexo的，为什么用Hexo那？

之前看过阮一峰老师的文章

github Pages和Jekyll入门

正打算用Jekyll搞，无意间在知乎上看到一条讨论

用 Jekyll 和 Octopress 搭建博客，哪个更合适？

于是发现了 Hexo，而且 Hexo 是基于 node.js ，就这点就立马吸引了我。

查阅资料，感谢 bruce 的文章 hexo你的博客 让我了解了hexo的基本用法和一些注意事项。

通过 Hexo 的官方文档了解到了 Hexo 的历史，不过发现 Hexo 的 主题并不是很多（地址），先用了 Noderce 的 Noderce Theme，之后找个时间自己也搞个把，算是贡献点自己的力量。

配置完多说后基本上就OK 了。

终于可以畅快的用markdown来写文章了。格式什么的都死一边去把。 - =

下一步的任务就是把之前的点滴记录都整理更新到这里了。

另外，启动一个自己的翻译计划：保持每2周翻译一篇英文技术方面的文章。

本周的第一篇译文已经更新到了BLOG。

为了自己的英文，这个一定要坚持下去。
